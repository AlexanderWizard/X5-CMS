# X5-CMS

> **Лёгкий модульный движок на Laravel 13 + Filament v4** — готовый фундамент для любого проекта вместо чистого листа.

X5-CMS собран из независимых модулей-папок (`API`, `CMS`, `System`), которые легко переносить и расширять одной строкой. Разворачивается за минуты и работает даже на **MySQL 5.6**.

## Возможности

- **Модульная архитектура** — самодостаточные модули, подключаются одной строкой.
- **CMS** — дерево страниц и шаблоны вывода прямо из базы: системные части (`header`, `menu`, `footer`), инклюды `@partial()` и SEO-поля для каждой страницы.
- **Тёмная админка Filament** с живым лендингом — контент-менеджер работает без правки вёрстки.
- **Безопасность из коробки** — роли с наглядным деревом прав по модулям, флаг суперпользователя, IP-файрвол и полный журнал действий администраторов.
- **Очередь сообщений** — приём через REST API с документацией **Swagger**, фоновая обработка по расписанию.
- **Мультиязычность** (ru/en) и автоматический часовой пояс пользователя.
- **SPA-навигация** — мгновенные переходы без перезагрузки страниц.
